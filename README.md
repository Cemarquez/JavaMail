# JavaMail
Uso de las librerías JavaMail con interfaz gráfica  y organizado para su fácil entendimiento.

[IMPORTANTE]
Si desean usar Gmail para enviar el correo deben permitir la opción de aplicaciones "menos seguras" ubicado aquí: https://myaccount.google.com/lesssecureapps. Una vez hecho esto, te permitirá iniciar sesión en la aplicación.
